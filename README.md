## Инструкция для сборки и запуска

Для работы с большими числами, нужна специальная библиотека Boost.
Её нужно установить руками.

git clone https://github.com/microsoft/vcpkg.git
./vcpkg/bootstrap-vcpkg.sh
cd ./vcpkg 
vcpkg.exe install boost-multiprecision
